Marble Maze

Created Zachary Kaatmann

For this project, my goal was to design an engaging, cheap, easily construct able marble game. These marble based game boards were to be sent to crisis relief shelters. For my project, I decided to opt with a 2 player game system, to keep the game play refreshing, and to give crisis victims a way to make new friends. This project taught me about the importance of using parameters to 3D model designs, as well as the relationship between real life engineering and designing a solution in Inventor. For all parts to fit in the correct spots, it was necessary to use parametric constraints to regulate the insert size. Any simple mistake in the 3D design process could cause the game to be be assemble-able. For more information, full documentation of this project can be found here.
