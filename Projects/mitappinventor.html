Mit App Inventor Project - Paint Pot

Created by Josh Tripathi and Zachary Kaatmann


After finishing the tutorial that laid the basic framework for the application. As a group, Josh and I decided to focus on adding more customization features for the end user. The application contains many features to select the color of your image modification, line size, and even the image displayed on the canvas. For those who aren't that creative or just don't want a picture of themselves on the tablet, there is a button labelled "random" that selects one of four images for you to draw on. To draw on the canvas, you can drag to create lines and tap to create dots. The size of both these elements can be adjusted by the sliders below. The data and camera functions allow for the user to take or select a picture to use for the canvas. The wipe button clears the canvas of all drawing, and the save button saves the canvas image to Internal Storage/Pictures/_app_inventor_image_picker.
